What makes it different

🎯

Base-rate anchoring

Every forecast starts from a historical prior computed independently per category — before any live signal is introduced — preventing baseline contamination.

📐

Per-category models

Supreme Court rulings and Senate cloture votes have structurally different outcome distributions. Eight separate logistic-regression classifiers respect that structure.

📊

Temporal calibration

Isotonic regression is fitted on a chronologically ordered held-out split. No look-ahead. Bootstrap CIs (1,000 resamples) quantify calibration uncertainty.

🔐

Signed audit trail

Every prediction is SHA-256 signed at write time. Append-only; nothing is editable after logging, making retrospective evaluation fully trustworthy.

Quick start

git clone https://github.com/OliverKlug/Uncertainty-aware-political-prediction.git
cd Uncertainty-aware-political-prediction
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env

# Run full daily pipeline
python src/pipeline/run_daily.py

# Backtesting
python src/backtesting/run_backtest.py --category all --window expanding

# Calibration report
python src/evaluation/calibration_report.py --output results/
